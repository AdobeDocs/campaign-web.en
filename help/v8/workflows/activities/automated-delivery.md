# Automated delivery {#automated-delivery}

>[!CONTEXTUALHELP]
>id="acw_orchestration_automated-delivery"
>title="Automated delivery activity"
>abstract="The **Automated delivery** activity lets you create, configure, and execute delivery actions within your workflow. You can create a new delivery from a template, select an existing explicit delivery that was created separately, or use a delivery specified in the inbound transition. Configure recipients and content, then choose the execution action.
